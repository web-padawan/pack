# Polymer Boilerplate 

## Installation 

`npm i -g gulp bower local-web-server` 

`npm i` 

`bower i` 

## Tasks 

### Serve project from /src 

`gulp` 

### Run production build 

`gulp build` 

### Serve project from /dist 

`gulp serve:dist` 

### Run JavaScript lint 

`npm run eslint -s`

### Run HTML hint 

`npm run htmlhint -s`
